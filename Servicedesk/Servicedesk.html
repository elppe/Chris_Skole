Lag din første github bruker og nettside

  1. Det første du trenger å gjøre for å lage en account på github er å gå til www.github.com å klikke på sign up oppe til høyre for å lage en bruker
  2. Så klikker du oppe til høyre igjen å klikker på your repositorys også klikker du new
  3. Velg et navn til repository og klikk "Add a README file" også klikker du på create repository
  4. Klikk på "Add file" også kall filen for "index.html". Dette vil være den første nettsiden som blir lastet når du går til nettsiden.
